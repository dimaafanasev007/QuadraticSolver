# Quadratic Equation Solver

Решение квадратных уравнений с длинной арифметикой.

## Структура проекта

Папка `src/Interpreter` — парсинг и проверка ввода  
Папка `src/BigNumber` — длинная арифметика  
Папка `src/Solver` — решение уравнения  
Папка `src/Result` — форматирование вывода  
Папка `tests` — модульные тесты

## Алгоритм

1. Интерпретатор проверяет входную строку (разделение по запятым, валидация чисел)
2. Длинная арифметика выполняет сложение, вычитание, умножение, деление, сравнение, квадратный корень
3. Решатель определяет тип уравнения:
   - Если a ≠ 0: квадратное, считаем дискриминант
   - Если a = 0 и b ≠ 0: линейное
   - Если a = 0 и b = 0 и c = 0: INF
   - Если a = 0 и b = 0 и c ≠ 0: NO SOLUTION
4. Результат выводится в формате OK, INF, WRONG или NO SOLUTION

## Сборка

mkdir build && cd build  
cmake ..  
cmake --build .

## Запуск

quadratic_solver.exe

## Тесты

solver_tests.exe

## Формат ввода

a,b,c (целые числа, разделённые запятыми)

## Примеры

**Два корня:**  
Вход: 1,-3,2  
Выход:  
OK  
2  
1  
2

**Один корень:**  
Вход: 1,-2,1  
Выход:  
OK  
1  
1

**Нет корней:**  
Вход: 1,0,1  
Выход: NO SOLUTION

**Линейное уравнение:**  
Вход: 0,2,-4  
Выход:  
OK  
1  
2

**Бесконечно много решений:**  
Вход: 0,0,0  
Выход: INF

**Нет решений (противоречие):**  
Вход: 0,0,5  
Выход: NO SOLUTION

**Неверный ввод (буквы):**  
Вход: 1,hello,3  
Выход: WRONG

**Неверный ввод (лишняя запятая):**  
Вход: 1,2,3,4  
Выход: WRONG

**Большие числа:**  
Вход: 12345678901234567890,0,0  
Выход:  
OK  
1  
0
