# QuadraticSolver

Решение квадратных уравнений ax² + bx + c = 0 с коэффициентами произвольной точности. Длинная арифметика написана с нуля на строках — без сторонних библиотек.

## Возможности

- Длинная арифметика (до 10⁶ цифр)
- Поддержка дробных чисел (0.5, -3.14)
- Поддержка научной нотации (1e+1000, 1E-500)
- Комплексные корни при D < 0
- Квадратный корень методом Ньютона (точность 60+ знаков)
- Google Test для модульного тестирования

## Сборка и запуск

```bash
cmake -B build
cmake --build build --config Release
./build/QuadraticSolver

## Тесты
run_tests.bat

## Формат ввода

Три числа через пробел: `a b c`

Поддерживаются:
- Целые числа: `42 -17 0`
- Дробные: `3.14 -0.5 .25`
- Научная нотация: `1e10 2.5e-3 1E-500`

## Формат вывода

- `OK` + количество корней + сами корни
- `INF` — бесконечно много решений
- `WRONG` — неверный ввод
- `NO SOLUTION` — нет решений
